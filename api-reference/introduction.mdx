## Authentication

All API endpoints are authenticated using Bearer tokens you can generate from your dashboard. To authenticate your requests, include the following header in your request:

```json
"Authorization": [
  {
    "X-Api-Key": [
      "example businfo-api"
    ]
  }
]
```
